Predicting Student Performance

Introduction: Student Performance Dataset

The Student Performance Dataset is from a study of secondary education of two Portuguese schools. The data was collected via school reports and questionnaires. G3 or the final grade is the main outcome variable for the dataset. The data was obtained from the UCI Machine Learning Repository and can be found at: http://archive.ics.uci.edu/ml/datasets/Student+Performance. The following table lists each variable and their descriptions.

Student Peformance Dataset Methods of Analysis:

Overview: For the Student Performance Dataset I utilized several modeling techniques (Simple Linear Regression, Multiple Linear Regression, Subset Selection, and Generalized Additive Models). Correlation models were not utilized as many of the predictors are qualitative while the outcome variable is quantitative. The utilization of the plot function helped to determine what variables in the dataset would be significant and also of those which would not be significant. While classification models (LDA, QDA, and KNN) are useful for this dataset they were only able to be utilized sparingly due to the continuous nature of the outcome variable. For several variables the outcome and predictor variables were switched with the outcome variable of G3 to get some sense of the predictive capabilities of the Logisitic Regression, QDA, and LDA modeling techniques. The classification models also did not allow for the interpretability of all of the variables for this particular dataset. KNN was ignored due to the fact it would over fit the data due to the small nature of the dataset itself
